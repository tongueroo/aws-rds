# AWS RDS Tool

Simple tool to create AWS RDS db instances with some pre-configured settings.

## Usage

```sh
aws-rds create my-db
```

## Settings

```sh
aws-rds create my-db --adapter=psql
```

```sh
aws-rds create my-db --profile=production
```

## Installation

```sh
gem install aws-rds
```
